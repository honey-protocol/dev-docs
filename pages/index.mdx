# Overview
### Integrations & Contributions
 
These developer docs are meant to help open source contributors build on top of the Honey protocol, or integrate it natively into their own applications.

# Use cases

- Leverage for long tail assets / NFTs
- Hedging the risk of risky assets
- Leasing / financial infrastructure for metaverse games
- Additional revenue stream for DAOs
- Instant liquidity for NFTs
